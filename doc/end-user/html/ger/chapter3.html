3. Bilderdatenbanken

3.1 Erstellen einer Bilderdatenbank

Vor dem Erstellen eines Fotomosaiks ist es notwendig, eine Bilderdatenbank zu erstellen, die die Bilder enthält, aus denen das Mosaik bestehen wird.


Klicken Sie dazu auf die "Datenbank erstellen"-Schaltfläche im Hauptfenster.


Im obigen Dialog müssen Sie einen Dateinamen für die neue Datenbank angeben.
Wählen Sie auch ein Verzeichnis, das die Dateien enthält, aus denen das Mosaik später bestehen soll. Beachten Sie, dass das Ergebnis je besser sein wird, desto mehr Bilder in der Datenbank sind. Das Hinzufgen weiterer Verzeichnisse und Dateien ist nach dem Erstellen im "Datenbank bearbeiten"-Dialog möglich. Um gute Ergebnisse zu erzielen, sollten sich mehrere Tausend Bilder mit verschiedenen Motiven und Grundfarben in der Datenbank befinden. Es ist auch von Vorteil, wenn alle Bilder das selbe oder ein ähnliches Seitenverhältnis aufweisen. Für die besten Ergebnisse sollte die Farbverteilung gleichmäßig und der Kontrast und die Dynamik der Bilder gering sein. Sie werden aber auch mit Bildern, die diese Kriterien nicht erfüllen, zu guten Ergebnissen gelangen - zur Not können Sie das durch Erhöhen der Anzahl der Kacheln im Mosaik ausgleichen. Wollen Sie auch die Bilder in den Unterverzeichnissen des gewählten Ordners hinzufügen, wählen Sie die entsprechende Option. Bestätigen Sie Ihre Einstellungen durch betätigen der "Datenbank erstellen"-Schaltfläche, um die Dateien im gewählten Verzeichnis zu analysieren und danach die Datenbankdatei zu speichern.


Haben Sie nun etwas Geduld, während die Datenbank erstellt wird. Abhängig von der Anzahl der Bilder im gewählten Verzeichnis und der Geschwindigkeit Ihres Rechners könnte dies eine Weile dauern. Zuerst werden die Datein im gewählten Verzeichnis indiziert und zur Datenbank hinzugefügt - danach werden Sie analysiert, um die Durchschnittsfarbe zu ermitteln.
Wenn Sie den Prozess während des Indizierens abbrechen, wird keine Datenbank gespeichert werden. Wenn Sie die Analyse der Dateien abbrechen, wird die Datenbank gespeichert werden, zum Erstellen eines Mosaiks werden aber nur die bereits analysierten Dateien verwendet werden können. Sie können die ausstehenden Dateien jedoch später im "Datenbank bearbeiten"-Dialog analysieren lassen.


Wenn Sie einen Dialog ähnlich dem folgenden sehen, ist die Datenbank nun bereit zur Benutzung und wurde erfolgreich gespeichert. Sie können nun Fotomosaiks damit erstellen or sie bearbeiten (Einträge hinzufügen und entfernen) - siehe den folgenden Abschnitt.

3.2 Bearbeiten einer Bilderdatenbank

Möchten Sie Bilder aus einer bestehenden Datenbank entfernen oder zu dieser hinzufügen oder sollten Probleme mit der Datenbank auftreten (etwa wenn Sie Dateien nach dem Hinzufügen zur Datenbank gelöscht haben), wählen Sie im Hauptfenster "Datenbank bearbeiten".
Wählen Sie nun die zu bearbeitende Datenbank aus.


Fall die Datenbank Dateien enthält, die noch nicht analysiert wurden, die seit der Analyse verändert wurden oder die unter dem angegebenen Pfad nicht zu finden sind, werden Sie den folgenden Dialog sehen. Wenn Sie mit "Ja" antworten, werden alle Dateien analysiert und die nicht gefunden markiert (prüfen Sie auch, ob diese sich auf einem nicht verbundenen Wechseldatenträger befinden!). Sie können dies auch später im "Datenbank bearbeiten"-Dialog nachholen. Beachten Sie, dass dies ein Muss ist, falls sich in der Datenbank Einträge zu Dateien befinden, die Sie in der Zwischenzeit gelöscht haben. Ansonsten wird das Erstellen eines Mosaiks mit dieser Datenbank fehlschlagen.


Dieser Dialog erlaubt Ihnen, die Datenbank zu bearbeiten.
Oben wird die Anzahl der Einträge in der Datenbank insgesamt und die Anzahl der noch nicht analysierten beziehungsweise nicht gefundenen Dateien angezeigt.
Links sehen Sie eine Liste mit allen Einträgen in der Datenbank. Wenn Sie einen auswählen, werden in der Mitte ein Vorschaubild und die durchschnittliche Farbe angezeigt. Mit der "Diese Datei aus der Datenbank entfernen"-Schaltfläche können Sie diese gewählte Datei aus der Datenbank löschen (Natürlich wird dabei die Datei auf dem Datenträger nicht gelöscht).
In der oberen rechten Ecke sehen Sie eine Liste mit allen Einträgen, die nicht gefunden werden konnten oder die keine gültigen Bilddateien sind. Mit der "Nicht gefundene Dateien entfernen"-Schaltfläche können Sie diese Einträge aus der Datenbank entfernen. Prüfen Sie zuerst, ob es sich um Verweise auf einen nicht eingelegten Wechseldatenträger handelt!
Darunter befindet sich eine Liste der noch nicht analysierten Bilder. Das bedeutet, dass diese Einträge auch nicht verwendet werden können, wenn Sie ein Mosaik auf Grundlage dieser Datenbank erstellen. Drücken Sie die "Datenbank aktualisieren"-Schaltfläche, um diese Dateien zu analysieren.
Mit den "Verzeichnis zur Datenbank hinzufügen"- und "Datei zur Datenbank hinzufügen"-Schaltflächen können Sie weitere Einträge hinzufgen.
Die Veränderungen in der Datenbank werden erst gespeichert, wenn Sie die "Schließen"-Schaltfläche betätigen und die folgende Frage mit "Ja" bestätigen.
Sollten Sie beim Erstellen eines Mosaiks mit dieser Datenbank Probleme haben, sollten Sie diese laden und die "Datenbank aktualisieren"-Schaltfläche betätigen. Damit werden nicht nur die ausstehenden Datein anaylsiert, es wird auch nach Fehlern in der Datenbank gesucht und versucht, diese zu korrigieren.
