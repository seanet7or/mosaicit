4. Erstellen eines Fotomosaiks

Wenn Sie bereits eine Bilderdatenbank erstellt haben, können Sie damit nun ein Fotomosaik erzeugen.


Klicken Sie dazu auf die "Neues Mosaik"-Scaltfläche im Hauptfenster.


Im folgenden Dialog müssen Sie die grundlegenden Einstellungen für das neue Mosaik treffen. Wählen Sie im "Datenbank"-Feld die Datenbank mit den Bildern für das Mosaik aus. Im "Ursprungsbild"-Feld müssen Sie das in ein Mosaik umzuwandelnde Bild angeben. Geben Sie in dem untersten Feld den Dateinamen, unter dem das neu erstellte Mosaik gespeichert werden soll, an. Betätigen Sie zum Abschluss die "Weiter"-Schaltfläche.


Jetzt können Sie die detaillierten Einstellungen für das neue Mosaik festlegen.
Zuerst die Größe der einzelnen Kacheln. Das Mosaik wird aus vielen kleinen Kacheln zusammengesetzt, jede mit der hier festgelegten Größe. Wenn Sie beabsichtigen, das Mosaik zu drucken, sollten Sie für die längere Seite einer Kachel mindestens 1 cm Länge einplanen. Wählen Sie die Kachelgröße dann entsprechend der gewünschten Auflösung. Werte im Bereich von 100-150 Pixel sind ein guter Startwert.
Darunter können Sie das Seitenverhältnis der Kacheln festlegen; Wählen Sie das Seitenverhältnis, das dem der meisten Bilder in der Datenbank entspricht. Sollten diese unterschiedliche Seitenverhältnisse haben (einige hochkant und einige querkant), so ist 1:1 eine gute Wahl.


Die nächste Option beschreibt, wie die Kacheln, deren Größenverhältnis nicht dem oben festgelegten entspricht, angepasst werden; Sie können entweder die Ränder beschneiden oder das Bild skalieren. Beachten Sie, dass wenn ein großer Unterschied zwischen dem ursprünglichen und dem von Ihnen gewählten Seitenverhältnis liegt, durch das Beschneiden der Ränder möglicherweise wichtige Details abgeschnitten werden könnten. Andererseits werden die Kacheln in diesem Fall durch das Skalieren verzerrt erscheinen. Deshalb werden Sie die besten Ergebnisse erreichen, wenn Sie Bilder wählen, bei denen sich keine wichtigen Details am Rand befinden (etwa Landschaften) oder wenn alle Bilder in der Datenbank ein ähnliches Seitenverhältnis haben. Ich empfehle hier, die Ränder zu beschneiden.
Legen Sie dann die Anzahl der Kacheln in dem Mosaik insgesamt fest; Je höher Sie diesen Wert wählen, desto besser und beeindruckender wird das Ergebnis, aber auch die benötigte Zeit und der benötigte Speicher zum Rendern sein. Beachten Sie auch, dass falls Sie das Mosaik etwa als Poster drucken möchten, Sie eine tatsächliche Länge von mindestens 1 cm für die längere Kachelseite wählen sollten. Sollten Sie also ein Mosaik mit 4800 Kacheln erstellen und das Ursprungsbild ein Seitenverhältnis von 4:3 haben mit einem Seitenverhältnis der Kacheln von 1:1, dann wird das Mosaik aus 80x60 Kacheln erstellt werden. Also müssten Sie ein Poster mit einer Größe von mindestens 80x60 cm drucken, um sicherzustellen, dass die einzelnen Kacheln noch erkennbar sind. Ich empfehle, hier einen Wert zwischen 2000 und 10000 zu wählen.
Die Option "Minimaler Abstand zwischen den selben Kacheln" lässt Sie festlegen, wieviel Mindestabstand zwischen zwei Vorkommen derselben Kachel liegen muss. Beachten Sie, dass ein zu hoher Wert bei einer zu kleinen Datenbank die Bildqualität mindern oder sogar das Erstellen des Mosaiks unmöglich machen wird, wenn keine passende Kachel mehr gefunden werden kann. Ich empfehle, hier einen Wert aus dem Bereich von 4-10 zu wählen, abhängig von dem Ursprungsbild und der gewählten Datenbank.
Jetzt können Sie festlegen, wie oft eine Kachel maximal in einem Mosaik auftauchen darf. Beachten Sie, dass ein zu niedriger Wert die Bildqualität mindern oder sogar das Erstellen des Mosaiks unmöglich machen wird, wenn keine passende Kachel gefunden werden kann. Ich empfehle, diese Option nicht zu verwenden und stattdessen einen Mindestabstand zwischen dem Auftauchen der selben Kachel festzulegen.
Sie können auch einen Alphawert festlegen, mit dem das Ursprungsbild über das Mosaik gelegt wird. Dies wird die Farbe der Kacheln beeinflussen und ein besseres Ergebnis bringen, wenn die Bilder in Ihrer Datenbank nicht optimal sind. Wählen Sie einen Wert von "0", um diese Option zu deaktivieren; "1" bedeutet maximale Transparenz, "99" minimale Transparenz. Ich empfehle einen Wert aus dem Bereich von 30-33.
Abschließend wird die Größe des Mosaiks angezeigt. Dies kann einen Hinweis darauf geben, wieviel Speicher und Zeit zum Rendern des Mosaiks benötigt werden wird. Ein Mosaik mit 60 Megapixeln und Alpha blending mit dem Ursprungsbild wird etwa 300MB Arbeitsspeicher benötigen und auf einem etwas älteren AMD Athlon XP mobile 2600+ etwa 10-15 Minuten dauern.
Wenn Sie alle Einstellungen getroffen haben, wählen Sie die "Rendern"-Schaltfläche.


Haben Sie nun etwas Geduld, während das Mosaik erstellt wird. Abhängig von den gewählten Einstellungen und der Geschwindigkeit Ihres Rechners, wird das Rendern einige Zeit in Anspruch nehmen.


Wenn Sie einen Dialog ähnlich dem folgenden sehen, wurde das Mosaik erfolgreich erstellt! Sollte das Erstellen fehlschlagen, finden Sie weitere Informationen unter "5.1 Das Rendern des Mosaiks schlägt fehl".


Sie finden das fertige Mosaik nun unter dem gewählten Dateinamen.
